AI Aggregation Engine

The core logic that ensures Eventa always has the latest events without manual input.

  • Async Scraper (BeautifulSoup4 + httpx)

    Non-blocking HTTP requests fetch HTML from top event sites. It uses rotated user-agents and handles pagination dynamically to extract raw DOM structures.

  • OpenAI Data Structuring

    Passes raw unstructured HTML strings to `gpt-3.5-turbo` or `gpt-4` with strict JSON schemas, automatically inferring titles, descriptions, prices, exact geographic coordinates (Lat/Lng), and standardized categories.

  • Deduplication & Sanitization Pipeline

    Before inserting into MongoDB, the backend checks for identical event titles/locations/dates to prevent duplicates, and maps external generic organizer names to "Hosted on [Platform]".
