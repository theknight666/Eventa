Frontend Edge-Layer

Powered by Next.js and deployed to the Vercel Global Edge Network. Utilizes Incremental Static Regeneration (ISR) to cache millions of event pages globally with millisecond Time-To-First-Byte (TTFB).

State & Fetching

SWR handles optimistic UI updates and aggressive client-side caching. Zustand manages complex global organizer multi-step forms.
