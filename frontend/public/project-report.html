This is not a simple website. Eventa is a heavily orchestrated, multi-tier microservice ecosystem. We built a proprietary AI ingestion pipeline, a military-grade API layer, and an impossibly fast edge-rendered client. Here is every single feature built.

1. Autonomous AI Scraper & Ingestion Pipeline

Asynchronous Multi-Threaded Workers

Built a non-blocking `httpx` and `BeautifulSoup4` pipeline capable of sweeping top event sites globally without hitting memory limits. Incorporates dynamic User-Agent rotation to evade bot-protection mechanisms.

OpenAI GPT-4 Integration Core

Raw, dirty HTML DOM structures are streamed to OpenAI with highly restricted JSON schema guardrails. The AI infers titles, extracts deep pricing tiers, and classifies events dynamically without human intervention.

Geospatial Data Normalization

The AI parses raw address strings and intelligently maps them to exact Latitude and Longitude coordinates, storing them in GeoJSON format required for our hyper-fast MongoDB 2dsphere indexing.

Intelligent Deduplication Algorithm

Custom logic prevents database bloat by calculating Levenshtein distances on event titles and matching exact time/date/location hashes to reject identical incoming events.

3. Microservice Backend & DB Architecture

Database Schema (MongoDB via Motor)

`events` Collection

Houses thousands of complex objects. We built strict 2dsphere geospatial indices required to power the `$near` algorithms for our dynamic location filtering UI.

`users` & `organizers` Collections

Architected a dual-table authentication approach. Users have saved arrays, while organizers possess verification statuses, slugs, and rich-text profile schemas.

`alerts` Subscriptions Collection

Engineered a highly optimized subscription table indexing user IDs to specific keywords and lat/lng ranges for rapid cross-referencing against new AI scraper entries.

Full Role-Based Portals

  • Super Admin Dashboard (`/admin`) Complete system oversight. Built custom tabbed UI with live filtering to approve events, grant blue verification checks to organizers, forcefully delete users, and feature specific events globally.
  • Organizer Dashboard (`/organizer`) A private SaaS portal. Integrated `recharts` for visual data of ticket sales. Built an image uploading event creation wizard. Integrated `html5-qrcode` scanner so organizers can scan attendee QR tickets directly via mobile browser.

Zero-Trust Security Layer

  • Stateless Authorization Tokens are issued via `pyjwt`, completely stateless, eliminating session bloat. Passwords hashed using bcrypt with dynamic salting.
  • DDoS & Brute Force Prevention Deployed Redis-backed `slowapi` rate limiters. Admin endpoints are strictly isolated via `ADMIN_SECRET` environment variables.
  • XSS & Payload Sanitization All HTML inputs from Organizers are deeply sanitized. Backend utilizes exact Pydantic typing to crash immediately if malicious types are injected.
